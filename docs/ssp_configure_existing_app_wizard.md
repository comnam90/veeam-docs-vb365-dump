# Configuring Microsoft Entra Application


To configure an existing Microsoft Entra application for Restore Portal, do the following:

1. [Launch the Configure Application wizard](ssp_create_new_app_1.md).
2. [Configure connection to Restore Portal](ssp_create_new_app_configure_connection.md).
3. [Configure Microsoft Entra application](ssp_create_new_app_2.md).
4. [Log in to Microsoft 365](ssp_create_new_app_3.md).
