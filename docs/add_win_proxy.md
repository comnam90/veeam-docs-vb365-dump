In this article

Before you start adding Windows-based backup proxy servers, check whether a target machine meets the system requirements. For more information, see [System Requirements](vbo_system_requirements.md#BackupProxy).

|  |
| --- |
| Note |
| Microsoft .NET and Windows PowerShell must be installed on a target machine in advance. |

To add a new Windows-based backup proxy server to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Proxy wizard](launch_new_backup_proxy_wizard.md).
2. [Select operating system type](vbo_proxy_os.md).
3. [Specify a backup proxy server address](vbo_proxy_server_address.md).
4. [Specify credentials](vbo_proxy_account.md).
5. [Select the service account](add_windows_proxy_service_acc.md).
