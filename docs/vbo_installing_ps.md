# Installing PowerShell Toolkit


This installation scenario allows you to install the Veeam Backup for Microsoft 365 PowerShell toolkit on a separate physical or virtual Windows-based machine. After completing the installation process, you will be able to run Veeam Backup for Microsoft 365 PowerShell sessions on this machine.

To install the Veeam Backup for Microsoft 365 PowerShell toolkit on a separate machine, do the following:

1. [Launch the installation wizard](vbo_install_launch_ps.md).
2. [Select the component to install](vbo_install_select_ps.md).
3. [Read and accept the license agreement](vbo_install_agreements_ps.md).
4. [Check the system configuration](vbo_install_configuration_check_ps.md).
5. [Review the default installation settings](vbo_install_default_settings_ps.md).
6. [Specify data location](vbo_install_data_location_ps.md).
7. [Begin the installation process](vbo_install_begin_ps.md).
