In this article

You can open backups created by a backup job.

To open backups created by a backup job, do the following:

1. Open the Organizations view.
2. In the inventory pane, select an organization.

|  |
| --- |
| Tip |
| You can also select the root Organizations node to see all backup and backup copy jobs that were created for all organizations added to the scope. |

1. In the preview pane, do one of the following:

* Select a backup job and click Explore on the ribbon and then select Explore latest <product> state of <date\_and\_time> or Explore <product> point-in-time state.

* Right-click a backup job and select Explore latest <product> state of <date\_and\_time> or Explore <product> point-in-time state.

where <product> is one of the following services: Exchange, SharePoint, OneDrive, or Teams.

1. Proceed to [Data Restore](vbo_data_restore.md).
