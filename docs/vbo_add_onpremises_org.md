In this article

To add on-premises Microsoft Exchange and on-premises Microsoft SharePoint organizations, do the following:

1. [Launch the Add Organization wizard](launch_add_on_premises_wizard_sp_ex.md).
2. [Select an organization deployment type](select_organization_deployment_type_exchange_sp_ex.md).
3. [Specify Microsoft Exchange connection settings](specify_connection_settings_exchange_both.md).
4. [Specify Microsoft SharePoint connection settings](specify_connection_settings_sharepoint_both.md).
5. [Finish the wizard](finish_wizard_sp_ex.md).
