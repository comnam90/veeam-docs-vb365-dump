To upgrade the Veeam Backup for Microsoft 365 REST API component on a separate machine, do the following:

1. [Launch the upgrade wizard](vbo_upgrade_launch_rest.md).
2. [Read and accept the license agreement](vbo_upgrade_agreements_rest.md).
3. [Check the system configuration](vbo_upgrade_configuration_check_rest.md).
4. [Review the default installation settings](vbo_upgrade_default_settings_rest.md).
5. [Begin the installation process](vbo_upgrade_begin_rest.md).
