# Enabling TLS Encryption on PostgreSQL Instance


To enhance security of the PostgreSQL instance, we recommend that you also enable the TLS encryption for the PostgreSQL instance traffic. For more information, see [this PostgreSQL article](https://www.postgresql.org/docs/current/runtime-config-connection.html#RUNTIME-CONFIG-CONNECTION-SSL).
