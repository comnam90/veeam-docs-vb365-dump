In this article

By default, the Veeam Backup for Microsoft 365 console is installed on the Veeam Backup for Microsoft 365 server automatically when you install the product. You do not need to install the console manually.

However, in addition to the default console, you can install the Veeam Backup for Microsoft 365 console on a dedicated machine to access the Veeam Backup for Microsoft 365 server and backup infrastructure components remotely. You can install as many remote consoles as you need.

This installation scenario allows you to install the Veeam Backup for Microsoft 365 console along with Veeam Explorers on a physical or virtual Windows-based machine. Veeam Explorers are the set of instruments that comes as part of Veeam Backup for Microsoft 365 and allows you to restore or export your data from backups.

Before you install the Veeam Backup for Microsoft 365 console, [check prerequisites](vbo_install_console_before_you_begin.md).

To install the Veeam Backup for Microsoft 365 console, do the following:

1. [Launch the installation wizard](vbo_install_launch_console.md).
2. [Select the component to install](vbo_install_select_console.md).
3. [Read and accept the license agreement](vbo_install_agreements_console.md).
4. [Check the system configuration](vbo_install_configuration_check_console.md).
5. [Review the default installation settings](vbo_install_default_settings_console.md).
6. [Specify data location](vbo_install_data_location_console.md).
7. [Begin the installation process](vbo_install_begin_console.md).
