In this article

You can use the cmdlets from this section to perform the following operations with Restore Portal:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBORestorePortalSettings](get-vborestoreportalsettings.md) | Returns connection settings to access Restore Portal. |
| [Set-VBORestorePortalSettings](set-vborestoreportalsettings.md) | Modifies the Restore Portal settings. |
| [New-VBORbacOperator](new-vborbacoperator.md) | Defines users or groups that will act as restore operators. |
| [New-VBORbacRoleItem](new-vborbacroleitem.md) | Defines Microsoft 365 organization users, groups, sites or teams that will be managed by restore operators or excluded. |
| [Get-VBOOperatorAuthenticationSettings](get-vbooperatorauthenticationsettings.md) | Returns restore operator authentication settings. |
| [Set-VBOOperatorAuthenticationSettings](set-vbooperatorauthenticationsettings.md) | Modifies restore operator authentication settings. |
| [Add-VBORbacRole](add-vborbacrole.md) | Adds restore operator roles to Veeam Backup for Microsoft 365. |
| [Get-VBORbacRole](get-vborbacrole.md) | Returns settings of restore operator roles. |
| [Set-VBORbacRole](set-vborbacrole.md) | Modifies settings of a restore operator role. |
| [Remove-VBORbacRole](remove-vborbacrole.md) | Removes restore operator roles from Veeam Backup for Microsoft 365. |
