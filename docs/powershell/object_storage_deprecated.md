# Object Storage Deprecated Cmdlets


In Veeam Backup for Microsoft 365 8, these cmdlets became deprecated.

You can use the cmdlets from this section to perform the following operations with object storage repositories:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBOAmazonS3CompatibleObjectStorageRepository](add-vboamazons3compatibleobjectstoragerepository.md) | Adds S3 Compatible object storage repository. |
| [Add-VBOAmazonS3ObjectStorageRepository](add-vboamazons3objectstoragerepository.md) | Adds Amazon S3 object storage repository. |
| [Add-VBOAzureBlobObjectStorageRepository](add-vboazureblobobjectstoragerepository.md) | Adds Azure Blob Storage. |
| [Get-VBOObjectStorageRepository](get-vboobjectstoragerepository.md) | Returns object storage repositories added to the Veeam Backup for Microsoft 365 infrastructure. |
| [Set-VBOObjectStorageRepository](set-vboobjectstoragerepository.md) | Modifies settings of an object storage repository. |
| [Remove-VBOObjectStorageRepository](remove-vboobjectstoragerepository.md) | Removes an object storage repository from the Veeam Backup for Microsoft 365 infrastructure. |
