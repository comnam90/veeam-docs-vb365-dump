# Job Sessions


You can use the cmdlet from this section to perform the following operation with backup or backup copy job sessions:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOJobSession](get-vbojobsession.md) | Returns the backup or backup copy job sessions. |
