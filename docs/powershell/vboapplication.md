# VBOApplication


Contains details about a Microsoft Entra application.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Microsoft Entra application ID. |
| DisplayName | String | Microsoft Entra application name. |
| Tags | String[] | Array of tags. |
