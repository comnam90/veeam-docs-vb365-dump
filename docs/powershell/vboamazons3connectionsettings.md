# VBOAmazonS3ConnectionSettings


Contains details about connection settings to Amazon S3 object storage repository.

| Property | Type | Description |
| --- | --- | --- |
| RegionType | VBOAmazonS3RegionType | Amazon S3 region type. Possible values:   * Global * USGovernment * China |
| ServiceType | VBOAmazonS3ServiceType | Service type. Possible values:   * AmazonS3 * AmazonS3Compatible |
