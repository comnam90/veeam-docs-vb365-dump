# VBOAzureBlobObjectStorageRepository


Contains details about Azure Blob Storage. See also [VBORepository](vborepository.md).

| Property | Type | Description |
| --- | --- | --- |
| Folder | [VBOAzureBlobFolder](vboazureblobfolder.md) | Details about an Azure Blob folder. |
| UseArchiverAppliance | Bool | If True, usage of the Azure archiver appliance is enabled. |
| ApplianceInfo | [VBOAzureBlobObjectStorageApplianceInfo](vboazureblobobjectstorageapplianceinfo.md) | Details about the Azure archiver appliance. See also [VBOAzureArchiverAppliance](vboazurearchiverappliance.md). |
