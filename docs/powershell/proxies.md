# Backup Proxy Servers


You can use the cmdlets from this section to perform the following operations with backup proxy servers:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOProxy](get-vboproxy.md) | Returns backup proxy servers added to the Veeam Backup for Microsoft 365 infrastructure. |
| [Add-VBOProxy](add-vboproxy.md) | Adds backup proxy servers. |
| [Set-VBOProxy](set-vboproxy.md) | Modifies settings of backup proxy servers. |
| [Sync-VBOProxy](sync-vboproxy.md) | Rescans backup proxy servers. |
| [Remove-VBOProxy](remove-vboproxy.md) | Removes backup proxy servers. |
| [Update-VBOProxy](update-vboproxy.md) | Upgrades backup proxy servers. |
| [New-VBOLinuxCredential](new-vbolinuxcredential.md) | Creates the Linux credentials record to connect to a Linux-based backup proxy server. |
