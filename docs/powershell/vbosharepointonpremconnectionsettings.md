# VBOSharePointOnPremConnectionSettings


Contains details about connection settings to access the Microsoft on-premises SharePoint organization.

| Property | Type | Description |
| --- | --- | --- |
| ServerPort | Ushort | Port number to connect to the on-premises Microsoft SharePoint server. |
