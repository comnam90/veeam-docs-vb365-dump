In this article

The following topics contain instructions on specifying and managing Microsoft Azure Blob Storage settings. You can use these settings to add Microsoft Azure Blob Storage as object storage repository to the Veeam Backup for Microsoft 365 infrastructure.

If you want to use the Azure archiver appliance when Veeam Backup for Microsoft 365 copies backed-up data from Azure Blob Storage to Azure Blob Storage Archive, you must add the Microsoft Azure service account.

* [Microsoft Azure Service Accounts](azure_service_accounts.md)
