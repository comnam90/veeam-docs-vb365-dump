In this article

You can use the cmdlets from this section to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Connect-VBOServer](connect-vboserver.md) | Connects to the Veeam Backup for Microsoft 365 server. |
| [Disconnect-VBOServer](disconnect-vboserver.md) | Ends current PowerShell session. |
| [Get-VBOServer](get-vboserver.md) | Returns information about the Veeam Backup for Microsoft 365 server. |
| [Set-VBOServer](set-vboserver.md) | Allows you to specify temporary storage for export and save operations. |
| [Set-VBOConfigurationParameter](set-vboconfigurationparameter.md) | Modifies configuration of the Veeam Backup for Microsoft 365 controller, backup proxy servers and backup proxy pools. |
| [Set-VBOPSQLDatabaseServerLimits](set-vbopsqldatabaseserverlimits.md) | Modifies settings of the PostgreSQL instance. |
| [Get-VBOVersion](get-vboversion.md) | Returns Veeam Backup for Microsoft 365 version. |
