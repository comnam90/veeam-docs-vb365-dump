# VBOAmazonS3ObjectStorageApplianceInfo


Contains details about the Amazon archiver appliance. See also [VBOAmazonS3ObjectStorageRepository](vboamazons3objectstoragerepository.md).

| Property | Type | Description |
| --- | --- | --- |
| InstanceType | String | Instance type that the archiver appliance uses. |
| VpcId | String | ID of the Amazon Virtual Private Cloud (Amazon VPC). |
| SubnetId | String | Subnet ID. |
| SecurityGroupId | String | Security group ID. |
| Port | Int | Port number. |
| IpRanges | String[] | Range of IPv4 addresses. |
