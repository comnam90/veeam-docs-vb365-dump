# VBOGlobalRetentionExclusion


Contains details about retention policy settings for contacts and calendars.

| Property | Type | Description |
| --- | --- | --- |
| SkipCalendar | Bool | If True, retention policy is not applied to calendars. |
| SkipContacts | Bool | If True, retention policy is not applied to contacts. |
