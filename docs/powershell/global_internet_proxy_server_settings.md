# Global Internet Proxy Server Settings


You can use the cmdlets from this section to perform the following operations with global internet proxy server settings:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOInternetProxySettings](get-vbointernetproxysettings.md) | Returns global internet proxy server settings. |
| [Set-VBOInternetProxySettings](set-vbointernetproxysettings.md) | Modifies global internet proxy server settings. |
