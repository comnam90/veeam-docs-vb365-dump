In this article

You can use the cmdlets from this section to perform the following operations with Amazon S3 storage account credentials:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBOAmazonS3Account](add-vboamazons3account.md) | Creates Amazon S3 storage account credentials. |
| [Get-VBOAmazonS3Account](get-vboamazons3account.md) | Returns Amazon S3 storage account credentials. |
| [Set-VBOAmazonS3Account](set-vboamazons3account.md) | Modifies Amazon S3 storage account credentials. |
| [Remove-VBOAmazonS3Account](remove-vboamazons3account.md) | Removes Amazon S3 storage account credentials. |
