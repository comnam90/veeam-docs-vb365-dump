# VBOAmazonSubnet


Contains details about a subnet.

| Property | Type | Description |
| --- | --- | --- |
| VPC | [VBOAmazonVpc](vboamazonvpc.md) | Details about the Amazon Virtual Private Cloud (Amazon VPC). |
| Id | String | Subnet ID. |
| Name | String | Subnet name. |
| CIDR | String | Range of IPv4 addresses for the subnet in the form of a Classless Inter-Domain Routing (CIDR) block. |
