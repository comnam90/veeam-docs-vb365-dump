# VBOAzureBlobFolder


Contains details about an Azure Blob folder.

| Property | Type | Description |
| --- | --- | --- |
| Container | [VBOAzureBlobContainer](vboazureblobcontainer.md) | Details about an Azure Blob container. |
| Name | String | Azure Blob folder name. |
| Path | String | Path to the Azure Blob folder. |
