# VBOAzureBlobConnectionSettings


Contains details about connection settings to Microsoft Azure Blob Storage.

| Property | Type | Description |
| --- | --- | --- |
| Account | [VBOAzureBlobAccount](vboazureblobaccount.md) | Details about Microsoft Azure Blob Storage account credentials. |
| RegionType | VBOAzureRegionType | Microsoft Entra region. Possible values:   * Global * Germany * China * Government |

Related Commands

* [New-VBOAzureBlobConnectionSettings](new-vboazureblobconnectionsettings.md)
* [Get-VBOAzureBlobContainer](get-vboazureblobcontainer.md)
