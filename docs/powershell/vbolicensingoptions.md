# VBOLicensingOptions


Contains licensing details. See also [VBOOrganization](vboorganization.md).

| Property | Type | Description |
| --- | --- | --- |
| LicensedUsersCount | Uint | Number of licensed Veeam Backup for Microsoft 365 users. |
| TrialUsersCount | Uint | Number of new user accounts. |
