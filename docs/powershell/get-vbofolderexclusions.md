Short Description

Returns folder exclusion settings.

Syntax

|  |
| --- |
| Get-VBOFolderExclusions [<CommonParameters>] |

Detailed Description

This cmdlet returns folder exclusion settings. Backup jobs will not process the specified mailbox data.
