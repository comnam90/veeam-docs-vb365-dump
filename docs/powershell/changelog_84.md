# v8.4 Changelog


This section describes new PowerShell cmdlets in Veeam Backup for Microsoft 365 8.4 comparing to 8.3.

New Cmdlets

New Cmdlets

| Cmdlet | Operation |
| [Export-VBOJobLog](export-vbojoblog.md) | Exports the Veeam Backup for Microsoft 365 job log files. |
| [Get-VBOVersionBackupOptions](get-vboversionbackupoptions.md) | Returns the version backup options for Microsoft organizations. |
| [Set-VBOVersionBackupOptions](set-vboversionbackupoptions.md) | Modifies the version backup options for Microsoft organizations. |
