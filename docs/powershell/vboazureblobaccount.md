# VBOAzureBlobAccount


Contains details about Microsoft Azure Blob Storage account credentials.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Microsoft Azure Blob Storage account ID. |
| Description | String | Microsoft Azure Blob Storage account description. |
| LastModified | DateTime | Date and time when the Microsoft Azure Blob Storage account was last modified. |
| Name | String | Microsoft Azure Blob login name. |
