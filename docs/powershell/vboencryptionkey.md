# VBOEncryptionKey


Contains details about an encryption key.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Encryption key ID. |
| Description | String | Encryption key description. |
| LastModified | DateTime | Date and time when the encryption key was last modified. |
