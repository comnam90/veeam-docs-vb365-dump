# Archiver Appliances


You can use the cmdlets from this section to perform the following operations with archiver appliances:

| Cmdlet | Operation |
| --- | --- |
| [New-VBOAmazonArchiverAppliance](new-vboamazonarchiverappliance.md) | Defines the Amazon archiver appliance settings. |
| [New-VBOAzureArchiverAppliance](new-vboazurearchiverappliance.md) | Defines the Azure archiver appliance settings. |
| [Get-VBOAmazonInstanceType](get-vboamazoninstancetype.md) | Returns an instance type that the Amazon archiver appliance will use. |
| [Get-VBOAmazonSecurityGroup](get-vboamazonsecuritygroup.md) | Returns a security group that will be associated with the Amazon archiver appliance. |
| [Get-VBOAmazonSubnet](get-vboamazonsubnet.md) | Returns the Amazon archiver appliance subnet. |
| [Get-VBOAmazonVPC](get-vboamazonvpc.md) | Returns the Amazon Virtual Private Cloud (Amazon VPC). |
| [Get-VBOAzureLocation](get-vboazurelocation.md) | Returns a Microsoft Entra region. |
| [Get-VBOAzureResourceGroup](get-vboazureresourcegroup.md) | Returns a resource group that will be associated with the Azure archiver appliance. |
| [Get-VBOAzureSubNet](get-vboazuresubnet.md) | Returns the Azure archiver appliance subnet. |
| [Get-VBOAzureVirtualMachineSize](get-vboazurevirtualmachinesize.md) | Returns the Azure archiver appliance size. |
| [Get-VBOAzureVirtualNetwork](get-vboazurevirtualnetwork.md) | Returns a network to which the Azure archiver appliance will be connected. |
