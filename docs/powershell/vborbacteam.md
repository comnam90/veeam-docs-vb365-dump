# VBORbacTeam


Contains details about a team that is managed by restore operators or is excluded. See also [VBORbacRoleItem](vborbacroleitem.md).

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Team ID. |
| DisplayName | String | Team display name. |
| Mail | String | Email address of the team. |
| Description | String | Description of the team. |
