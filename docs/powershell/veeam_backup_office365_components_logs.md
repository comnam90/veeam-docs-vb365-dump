In this article

You can use the cmdlets from this section to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOServerComponents](get-vboservercomponents.md) | Returns components of the Veeam Backup for Microsoft 365 server. |
| [Set-VBOServerComponents](set-vboservercomponents.md) | Enables or disables the extended logging mode and distributed tracing for an array of the Veeam Backup for Microsoft 365 server components. |
| [Export-VBOLog](export-vbolog.md) | Exports the Veeam Backup for Microsoft 365 server log files. |
