# VBORestoreLogItem


Contains details about a log item. See also [VBORestoreSession](vborestoresession.md).

| Property | Type | Description |
| --- | --- | --- |
| ItemName | String | Name of the restored item. |
| ItemType | String | Type of the restored item. |
| ItemSize | Ulong | Size of the restored item. |
| SourceName | String | Source name. |
| TargetName | String | Target name. |
