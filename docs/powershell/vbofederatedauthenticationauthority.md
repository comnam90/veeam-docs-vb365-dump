# VBOFederatedAuthenticationAuthority


Contains details about an external authentication provider.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | External authentication provider ID. |
| Name | String | Name of the external authentication provider. |
| Description | String | Description of the external authentication provider. |
| Issuer | String | Issuer of the authorization token. |
