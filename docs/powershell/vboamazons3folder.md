# VBOAmazonS3Folder


Contains details about an Amazon S3 folder.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Amazon S3 folder name. |
| Path | String | Path to the Amazon S3 folder. |
| Bucket | [VBOAmazonS3Bucket](vboamazons3bucket.md) | Details about an Amazon S3 bucket. |
