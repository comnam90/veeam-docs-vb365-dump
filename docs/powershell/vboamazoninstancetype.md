# VBOAmazonInstanceType


Contains details about an instance type that the Amazon archiver appliance uses.

| Property | Type | Description |
| --- | --- | --- |
| Type | String | EC2 instance type. |
| Cores | Int | Number of processor cores. |
| Memory | Long | Amount of RAM. |
