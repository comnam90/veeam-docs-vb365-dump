# VBOS3ConnectionSettings


Contains details about connection settings to Amazon S3 object storage repository or S3 Compatible object storage repository. See also [VBOAmazonS3Bucket](vboamazons3bucket.md).

| Property | Type | Description |
| --- | --- | --- |
| ServiceType | VBOAmazonS3ServiceType | Service type. Possible values:   * AmazonS3 * AmazonS3Compatible |
| Account | [VBOS3Account](vbos3account.md) | Details about Amazon S3 storage account. |
