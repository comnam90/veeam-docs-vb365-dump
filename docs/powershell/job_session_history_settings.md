# Job Session History Settings


You can use the cmdlets from this section to perform the following operations with the history settings of backup and backup copy job sessions:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOHistorySettings](get-vbohistorysettings.md) | Returns job session history settings. |
| [Set-VBOHistorySettings](set-vbohistorysettings.md) | Modifies job session history settings. |
