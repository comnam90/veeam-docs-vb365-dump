# VBORepositoryUpgradeSession


Contains details about an upgrade session.

| Property | Type | Description |
| --- | --- | --- |
| RepositoryId | GUID | Backup repository ID. |
| RepositoryName | String | Backup repository name. |
| Status | String | Session status. |
