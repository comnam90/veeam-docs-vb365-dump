# VBOS3Account


Contains details about Amazon S3 storage account. See also [VBOS3ConnectionSettings](vbos3connectionsettings.md).

| Property | Type | Description |
| --- | --- | --- |
| ServiceType | VBOAmazonS3ServiceType | Service type. Possible values:   * AmazonS3 * AmazonS3Compatible |
| Id | GUID | Amazon S3 storage account ID. |
| Description | String | Amazon S3 storage account description. |
| LastModified | DateTime | Date and time when Amazon S3 storage account was last modified. |
| AccessKey | String | Access key. |
