In this article

You can use the cmdlets from this section to perform the following operations with connection settings:

| Cmdlet | Operation |
| --- | --- |
| [New-VBOOffice365ConnectionSettings](new-vbooffice365connectionsettings.md) | Defines connection settings to access Microsoft 365 organizations. |
| [Set-VBOOffice365ConnectionSettings](set-vbooffice365connectionsettings.md) | Modifies connection settings to access Microsoft 365 organizations. |
| [New-VBOOffice365ApplicationOnlyConnectionSettings](new-vbooffice365applicationonlyconnectionsettings.md) | Defines Microsoft Entra application settings. |
| [Set-VBOOffice365ApplicationOnlyConnectionSettings](set-vbooffice365applicationonlyconnectionsettings.md) | Modifies Microsoft Entra application settings. |
| [New-VBOOnPremConnectionSettings](new-vboonpremconnectionsettings.md) | Defines connection settings to access on-premises Microsoft organizations. |
| [Set-VBOOnPremConnectionSettings](set-vboonpremconnectionsettings.md) | Modifies connection settings to access on-premises Microsoft organizations. |
| [New-VBOBackupAccount](new-vbobackupaccount.md) | Creates auxiliary backup accounts to back up SharePoint and OneDrive for Business. |
| [Get-VBOApplication](get-vboapplication.md) | Returns Microsoft Entra applications added to Microsoft 365 organizations. |
| [New-VBOBackupApplication](new-vbobackupapplication.md) | Defines backup application settings to back up SharePoint and OneDrive for Business. |
| [Add-VBOBackupApplication](add-vbobackupapplication.md) | Adds backup application settings to an application pool. |
| [Get-VBOBackupApplication](get-vbobackupapplication.md) | Returns backup application settings that are added to Microsoft organizations. |
| [Set-VBOBackupApplication](set-vbobackupapplication.md) | Modifies backup application settings that are added to Microsoft organizations. |
| [Remove-VBOBackupApplication](remove-vbobackupapplication.md) | Removes backup application settings from Microsoft organizations. |
