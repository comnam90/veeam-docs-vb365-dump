# Restore Sessions


You can use the cmdlets from this section to perform the following operations with Veeam Backup for Microsoft 365 restore sessions:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBORestoreSession](get-vborestoresession.md) | Returns Veeam Backup for Microsoft 365 restore sessions. |
| [Stop-VBORestoreSession](stop-vborestoresession.md) | Stops Veeam Backup for Microsoft 365 restore sessions. |
