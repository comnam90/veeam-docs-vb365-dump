# Organization Objects


You can use the cmdlets from this section to perform the following operations with organization objects:

| Cmdlet | Operation |
| --- | --- |
| [New-VBOBackupItem](new-vbobackupitem.md) | Creates organization objects. |
| [Add-VBOBackupItem](add-vbobackupitem.md) | Adds a list of objects to a backup job. |
| [Get-VBOBackupItem](get-vbobackupitem.md) | Returns a list of objects included in a backup job. |
| [Set-VBOBackupItem](set-vbobackupitem.md) | Modifies organization object settings. |
| [Remove-VBOBackupItem](remove-vbobackupitem.md) | Removes an object from a backup job. |
| [Get-VBOExcludedBackupitem](get-vboexcludedbackupitem.md) | Returns a list of objects excluded from a backup job. |
| [Add-VBOExcludedBackupItem](add-vboexcludedbackupitem.md) | Creates a list of objects excluded from a backup job. |
| [Remove-VBOExcludedBackupItem](remove-vboexcludedbackupitem.md) | Removes objects from a list of exclusions specified for a backup job. |
| [Add-VBOOrganizationRetentionExclusion](add-vboorganizationretentionexclusion.md) | Adds retention policy settings for contacts and calendars of a specific organization. |
| [Get-VBOOrganizationRetentionExclusion](get-vboorganizationretentionexclusion.md) | Returns retention policy settings for contacts and calendars of a specific organization. |
| [Set-VBOOrganizationRetentionExclusion](set-vboorganizationretentionexclusion.md) | Modifies retention policy settings for contacts and calendars of a specific organization. |
| [Remove-VBOOrganizationRetentionExclusion](remove-vboorganizationretentionexclusion.md) | Removes retention policy settings for contacts and calendars of a specific organization. |
