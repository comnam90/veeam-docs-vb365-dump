You can use the cmdlets from this section to perform the following operations with the Veeam Backup for Microsoft 365 tenant authentication settings:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOTenantAuthenticationSettings](get-vbotenantauthenticationsettings.md) | Returns the Veeam Backup for Microsoft 365 tenant authentication settings. |
| [Set-VBOTenantAuthenticationSettings](set-vbotenantauthenticationsettings.md) | Modifies the Veeam Backup for Microsoft 365 tenant authentication settings. |
