In this article

You can use the cmdlets from this section to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBOLicense](get-vbolicense.md) | Provides information on the currently installed license for Veeam Backup for Microsoft 365. |
| [Install-VBOLicense](install-vbolicense.md) | Installs the license for Veeam Backup for Microsoft 365. |
| [Update-VBOLicense](update-vbolicense.md) | Updates the currently installed license for Veeam Backup for Microsoft 365. |
| [Uninstall-VBOLicense](uninstall-vbolicense.md) | Uninstalls the currently installed license from Veeam Backup for Microsoft 365. |
| [Get-VBOLicensedUser](get-vbolicenseduser.md) | Returns licensed Veeam Backup for Microsoft 365 users. |
| [Remove-VBOLicensedUser](remove-vbolicenseduser.md) | Removes the product license from Veeam Backup for Microsoft 365 users. |
