# VBOBackupApplication


Contains details about a backup application.

| Property | Type | Description |
| --- | --- | --- |
| OrganizationId | GUID | Microsoft organization ID. |
| ApplicationId | GUID | Backup application ID. |
| DisplayName | String | Display name of the backup application. |
| ApplicationCertificateThumbprint | String | Thumbprint of the backup application. |
