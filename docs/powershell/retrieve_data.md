# Data Retrieval


You can use the cmdlets from this section to perform the following operations with data retrieval:

| Cmdlet | Operation |
| --- | --- |
| [Start-VBODataRetrieval](start-vbodataretrieval.md) | Creates and starts retrieval jobs. |
| [Get-VBODataRetrieval](get-vbodataretrieval.md) | Returns retrieval jobs. |
| [Set-VBODataRetrieval](set-vbodataretrieval.md) | Modifies retrieval job settings. |
| [Get-VBODataRetrievalObject](get-vbodataretrievalobject.md) | Returns objects whose backed-up data was retrieved from an object storage repository by retrieval job. |
| [Get-VBODataRetrievalSession](get-vbodataretrievalsession.md) | Returns data retrieval sessions. |
