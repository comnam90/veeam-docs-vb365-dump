# VBOOrganizationTeam


Contains details about an organization team.

| Property | Type | Description |
| --- | --- | --- |
| OrganizationId | GUID | Microsoft organization ID. |
| OfficeId | GUID | ID of the team in the Microsoft 365 organization. |
| DisplayName | String | Display name of the team. |
| Description | String | Description of the team. |
| Mail | String | Email address of the team. |
