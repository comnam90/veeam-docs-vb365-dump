In this article

You can use the cmdlets from this section to perform the following operations with Microsoft organizations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBOOrganization](add-vboorganization.md) | Adds Microsoft organizations to the Veeam Backup for Microsoft 365 infrastructure. |
| [Get-VBOOrganization](get-vboorganization.md) | Returns Microsoft organizations added to the Veeam Backup for Microsoft 365 infrastructure. |
| [Set-VBOOrganization](set-vboorganization.md) | Modifies Microsoft organization settings. |
| [Remove-VBOOrganization](remove-vboorganization.md) | Removes Microsoft organizations from the Veeam Backup for Microsoft 365 infrastructure. |
| [Get-VBOOrganizationSynchronizationState](get-vboorganizationsynchronizationstate.md) | Returns status of synchronization of Microsoft organization objects with the organization cache database. |
| [Start-VBOOrganizationSynchronization](start-vboorganizationsynchronization.md) | Starts to synchronize Microsoft organization objects with the organization cache database. |
