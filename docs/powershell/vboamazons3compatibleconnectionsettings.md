# VBOAmazonS3CompatibleConnectionSettings


Contains details about connection settings to S3 Compatible object storage repository.

| Property | Type | Description |
| --- | --- | --- |
| ServiceType | VBOAmazonS3ServiceType | Service type. Possible values:   * AmazonS3 * AmazonS3Compatible |
| ServicePoint | String | S3 Compatible endpoint address. |
| CustomRegionId | String | Region ID. |
