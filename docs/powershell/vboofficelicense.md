# VBOOfficeLicense


Contains details about a license assigned to the organization user. See also [VBOOrganizationUser](vboorganizationuser.md).

| Property | Type | Description |
| --- | --- | --- |
| SkuId | GUID | Unique ID of the service SKU. |
| SkuPartNumber | String | Unique display name of the service SKU. |
