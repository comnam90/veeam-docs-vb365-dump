In this article

To add a new S3 Compatible object storage repository to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Repository wizard](launch_new_backup_repository_wizard_s3_compatible.md).
2. [Specify an object storage repository name](s3_object_storage_name.md).
3. [Select target location for backups](s3_target_location.md).
4. [Select object storage type](s3_object_storage_type.md).
5. [Specify object storage service point and account](s3_account.md).
6. [Specify object storage bucket](s3_object_storage_bucket.md).
7. [Specify a backup proxy server or a backup proxy pool](new_s3_backup_proxy.md).
8. [Specify retention policy settings](new_s3_retention.md).
9. [Enable data immutability and encryption](new_s3_configure_immutability.md).
