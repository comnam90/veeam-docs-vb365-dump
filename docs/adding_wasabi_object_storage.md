# Adding Wasabi Cloud Object Storage Repositories


To add a new Wasabi Cloud Object Storage to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Repository wizard](launch_new_backup_repository_wizard_wasabi.md).
2. [Specify an object storage repository name](wasabi_repository_name.md).
3. [Select target location for backups](wasabi_target_location.md).
4. [Select object storage type](wasabi_storage_type.md).
5. [Specify object storage account](wasabi_account_and_point.md).
6. [Specify object storage bucket](wasabi_storage_settings.md).
7. [Specify a backup proxy server or a backup proxy pool](new_wasabi_backup_proxy.md).
8. [Specify retention policy settings](new_wasabi_retention.md).
9. [Enable data immutability and encryption](new_wasabi_configure_immutability.md).
