In this article

This installation scenario allows you to install Veeam Explorers when configuring Backup as a Service with Veeam Backup for Microsoft 365 for tenants. For more information, see [For Tenants](vbo_baas_tenant.md).

|  |
| --- |
| Note |
| You can install Veeam Explorers individually only on a server running Veeam Backup & Replication. |

To install Veeam Explorers on a server with Veeam Backup & Replication, do the following:

1. [Launch the installation wizard](vbo_install_launch_ve.md).
2. [Select the component to install](vbo_install_select_ve.md).
3. [Read and accept the license agreement](vbo_install_agreements_ve.md).
4. [Check the system configuration](vbo_install_configuration_check_ve.md).
5. [Begin the installation process](vbo_install_begin_ve.md).
