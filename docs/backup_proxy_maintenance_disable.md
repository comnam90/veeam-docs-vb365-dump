# Returning From Maintenance


To return backup proxy servers from maintenance, do the following:

1. [Launch the Proxy Maintenance wizard](launch_new_proxy_maintenance_wizard_disable.md).
2. [Select a backup proxy server](vbo_selecting_proxy_server_maintenance_disable.md).
3. [Specify credentials](vbo_maintenance_creds_disable.md).
4. [Specify the SSH connection settings](vbo_maintenance_ssh_disable.md).
5. [Review the operation summary](vbo_maintenance_summary_disable.md).
