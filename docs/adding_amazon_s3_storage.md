In this article

To add a new Amazon S3 object storage repository to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Repository wizard](launch_new_backup_repository_wizard_amazon.md).
2. [Specify an object storage repository name](amazon_object_storage_name.md).
3. [Select target location for backups](amazon_target_location.md).
4. [Select object storage type](amazon_object_storage_type.md).
5. [Select Amazon S3 storage type and account](amazon_storage_class.md).
6. [Specify the object storage repository settings](amazon_object_storage_settings.md).
7. [Select Amazon S3 Glacier storage class](amazon_glacier_storage_class.md).
8. [Specify a backup proxy server or a backup proxy pool](new_amazon_s3_backup_proxy.md).
9. [Specify retention policy settings](new_amazon_s3_retention.md).
10. [Enable data immutability and encryption](new_amazon_s3_configure_immutability.md).
