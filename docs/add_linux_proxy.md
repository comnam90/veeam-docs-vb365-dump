# Adding Linux-Based Backup Proxy Server


Before you start adding Linux-based backup proxy servers, check whether a target machine meets the system requirements. For more information, see [System Requirements](vbo_system_requirements.md#BackupProxy).

Veeam Backup for Microsoft 365 supports adding Linux-based backup proxy servers that can be non-domain and domain.

To add a new Linux-based backup proxy server to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Proxy wizard](launch_new_backup_proxy_wizard_linux.md).
2. [Select operating system type](vbo_proxy_os_linux.md).
3. [Specify a backup proxy server address](vbo_proxy_server_address_linux.md).
4. [Specify the SSH connection settings](add_linux_proxy_ssh.md).
5. [Select the service account](add_linux_proxy_service_acc.md).
