In this article

To add a new Microsoft Azure Blob Storage to the Veeam Backup for Microsoft 365 backup infrastructure, do the following:

1. [Launch the New Backup Repository wizard](launch_new_backup_repository_wizard_azure.md).
2. [Specify an object storage repository name](azure_object_storage_name.md).
3. [Select target location for backups](azure_target_location.md).
4. [Select object storage type](azure_object_storage_type.md).
5. [Select Microsoft Azure Blob storage type and account](azure_blob_storage_type.md).
6. [Specify object storage container](azure_object_storage_settings.md).
7. [Specify a backup proxy server or a backup proxy pool](new_azure_backup_proxy.md).
8. [Specify retention policy settings](new_azure_retention.md).
9. [Enable data immutability and encryption](new_azure_configure_immutability.md).
