In this article

This step is only available if you have selected to configure installation settings manually at the [Review Default Installation Settings](vbo_install_default_settings_rest.md) step of the wizard.

At the Data Location step, specify the Veeam Backup for Microsoft 365 REST API component installation folder.

By default, the Veeam Backup for Microsoft 365 REST API component is installed to the C:\Program Files\Veeam\Backup365 folder. To install to a different location, click Browse and specify a folder.
