# Creating Retrieval Job


To create a retrieval job, do the following:

1. [Launch the Retrieve Backup Copy wizard](launch_retrieve_wizard.md).
2. [Specify a retrieval job name](specify_retrieval_job_name.md).
3. [Specify point in time](retrieval_point_in_time.md).
4. [Select an organization](retrieval_organization.md).
5. [Select objects](retrieval_select_objects.md).
6. [Select a retrieval mode](retrieval_mode.md).
7. [Specify data availability period](availability_period.md).
