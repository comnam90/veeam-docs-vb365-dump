In this article

This step is only available if you have selected to configure installation settings manually at the [Review Default Installation Settings](vbo_install_default_settings_console.md) step of the wizard.

At the Data Location step, specify the Veeam Backup for Microsoft 365 console installation folder.

By default, the Veeam Backup for Microsoft 365 console is installed to the C:\Program Files\Veeam\Backup365 folder. To install to a different location, click Browse and specify a folder.

|  |
| --- |
| Note |
| Veeam Explorers are installed to the C:\Program Files\Veeam\Backup and Replication\Explorers folder. You cannot specify a different location for Veeam Explorers. |
