# Enabling TLS Encryption on NATS Server


By default, the NATS server that you deploy along with Veeam Backup for Microsoft 365 on the target machine does not use the TLS encryption.

To enhance security of the NATS server, we recommend that you enable the TLS encryption for the NATS server traffic. For more information, see [this NATS article](https://docs.nats.io/running-a-nats-service/configuration/securing_nats/tls).
