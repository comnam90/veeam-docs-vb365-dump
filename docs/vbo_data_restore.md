In this article

To restore Microsoft organization data, you can use Veeam Explorers and Restore Portal:

* [Veeam Explorer for Microsoft Exchange](https://helpcenter.veeam.com/docs/vbo365/explorers/vex_user_guide.html?ver=80)

To explore and restore Microsoft Exchange mailboxes, folders, messages, tasks, contacts and items.

* [Veeam Explorer for Microsoft SharePoint](https://helpcenter.veeam.com/docs/vbo365/explorers/vesp_user_guide.html?ver=80)

To explore and restore Microsoft SharePoint sites, libraries and items.

* [Veeam Explorer for Microsoft OneDrive for Business](https://helpcenter.veeam.com/docs/vbo365/explorers/veod_user_guide.html?ver=80)

To explore and restore Microsoft OneDrive for Business items and folders.

* [Veeam Explorer for Microsoft Teams](https://helpcenter.veeam.com/docs/vbo365/explorers/vet_user_guide.html?ver=80)

To explore and restore Microsoft Teams teams, channels, tabs, posts and files.

* [Restore Portal](ssp_restore.md)

To explore and restore Microsoft Exchange, Microsoft SharePoint, Microsoft OneDrive for Business and Microsoft Teams data using Restore Portal. Restore Portal allows users to perform self-service restore.

| Tip |
| When you use Veeam Explorers to explore backups, you can search for required items in a backup file. In particular, you can use the Advanced Find mechanism that allows you to configure search criteria using filters. For more information, see the Searching for Objects in Backup File sections of the [Veeam Explorers User Guide](https://helpcenter.veeam.com/docs/vbo365/explorers/explorers_introduction.html?ver=80) for each Veeam Explorer. |

To launch Veeam Explorers, you use the Explore option. For more information, see the following sections:

* [Exploring Backup Jobs](vbo_exploring_backup_jobs.md)

To open backups created by the selected backup job.

* [Exploring Single Organization](vbo_exploring_single_organization.md)

To open backups created by all backup jobs of a specific organization.

* [Exploring All Organizations](vbo_exploring_all_organizations.md)

To open backups of all organizations.

* [Exploring Retrieved Data](vbo_exploring_retrieved.md)

To open backups that you have retrieved from backup copies located in Azure Blob Storage Archive access tier, Amazon S3 Glacier Flexible Retrieval or Amazon S3 Glacier Deep Archive storage classes.

* [Exploring Backup Copies](vbo_exploring_backup_copies.md)

To open backup copies stored in Azure Blob Storage, Amazon S3 Standard, Amazon S3 Standard-Infrequent Access, Amazon S3 One Zone-Infrequent Access and Amazon S3 Glacier Instant Retrieval storage classes or S3 Compatible object storage repositories.

You can also restore Microsoft organization data from backups created for the Veeam Backup for Microsoft 365 server by Veeam Backup & Replication. For more information, see the [Application Item Restore](https://helpcenter.veeam.com/docs/vbr/userguide/restore_veeam_explorers.html?ver=13) section of the Veeam Backup & Replication User Guide.
